# React 建立開發環境練習

## 環境需求

+ node: `^8.9.3`
+ npm: `^6.8.0`

## 使用方式

```sh
$ git clone https://github.com/PenguinRun/goodidea-react-track.git
$ cd goodidea-react-track
$ npm install
```

| 指令 | 說明                                                                         |
|--------------------|----------------------------------------------------------------------------|
| `npm start`        | 啟動網站在`3000`port                                                         |
| `npm run deploy`   | 將整個專案打包到`dist`資料夾底下 |
